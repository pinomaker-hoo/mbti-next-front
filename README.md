## MBTI FrontEnd Repository

### URL

```
http://mbti.pinodev.shop:3000/

```

### Node, Npm Version

```
node : 16.18.0
npm : 6.14.15
```

### useing Stack

```
Next, Redux Toolkit, Redux Toolkit Query, Mui
```

### Project Start

```
1. git clone

2. yarn install

3. yarn dev
```

### Project Deploy

```
1. git clone

2. yarn

3. yarn build

4. pm2 start yarn -w -i 0 --name "next" -- start
```
